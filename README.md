# OOPS_Project
Fest management application with AI prediction tool 

Implementation plan: [IMPLEMENTATION_PLAN.md](IMPLEMENTATION_PLAN.md)

Project entry points:

- Build and run with [mvnw.ps1](mvnw.ps1) or [mvnw.cmd](mvnw.cmd)
- Open the dashboard at [src/main/resources/static/index.html](src/main/resources/static/index.html)
- Use the sample requests in [api-examples.http](api-examples.http)
- Read PostgreSQL setup notes in [POSTGRESQL_SETUP.md](POSTGRESQL_SETUP.md)
